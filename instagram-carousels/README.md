# Instagram Carousels — SunVault

Two AI-generated Instagram carousel sets (nano-banana via kie.ai), built from `../brief.md`
using the `../carousel-generator` skill, with the real SunVault logo composited onto each slide.
4:5 portrait, 1080×1350, on-brand navy + amber + live-green.

## Set 1 — The Blind Spot (`set-1-blind-spot/`)
Wake-up-call angle: most solar owners never see if their system works.
`slide-1.png … slide-4.png` + `caption.md`

## Set 2 — Luxury Solar That Thinks (`set-2-luxury-thinks/`)
Differentiator angle: one dedicated engineer + lifetime monitoring.
`slide-1.png … slide-4.png` + `caption.md`

## How to post
1. Instagram → new post → upload the 4 slides of a set **in order**.
2. Paste the caption from that set's `caption.md`.
3. Put your landing link (https://sun-vault-ashen.vercel.app/) in your bio.
4. Add music before posting for a Reels-tab reach boost.

## Also usable as Facebook ad creative
These slides can be used as the carousel creative for the Part 3 Facebook ad (draft),
alongside or instead of the crisp HTML-rendered set in `../carousel/`.
